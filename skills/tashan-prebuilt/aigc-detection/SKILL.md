# AIGC 痕迹检测

这个 skill 只做编辑判断，不做司法或取证意义上的“AI 生成鉴定”。输出必须用风险和证据表达，不能宣称百分百由 AI 生成。

## 检查维度

1. 句式：并列排比过多、连接词机械、段落开头同质化。
2. 内容：空泛拔高、缺少具体对象、缺少动作细节、只讲价值不讲证据。
3. 语气：助手口吻、营销腔、模板化总结、过强确定性。
4. 逻辑：跳过中间论证、把建议写成事实、把相关写成因果。
5. 细节：不自然的术语堆叠、泛化案例、无来源数字或引用。

## 工作流

1. 先说明检测边界：这是编辑风险评估，不是取证检测。
2. 按段落标出风险等级：高、中、低。
3. 引用短句作为证据，每条证据只摘必要片段。
4. 给出改写策略：补具体对象、换成人的判断顺序、拆开空泛句、降级强结论。
5. 如用户要求，输出一版保真改写；事实、数字、引用、专名和证据强度不得改变。

## 输出格式

- 总体风险：高 / 中 / 低。
- 主要 AI 痕迹：按证据列出。
- 段落级建议：段落编号、问题、修改方向。
- 保真改写：只在用户需要时提供。
- 不能判断的部分：材料不足时明确说明。
