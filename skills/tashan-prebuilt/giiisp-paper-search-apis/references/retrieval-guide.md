## 1. 知网（CNKI）高级检索语法

### 字段标识符

| 标识符 | 含义 | 示例 |
|--------|------|------|
| `SU=` | 主题 | `SU=人工智能` |
| `TI=` | 题名 | `TI=深度学习` |
| `KY=` | 关键词 | `KY=神经网络` |
| `AB=` | 摘要 | `AB=自然语言处理` |
| `AU=` | 作者 | `AU=张三` |
| `AF=` | 机构 | `AF=北京大学` |
| `YE=` | 年份 | `YE=2020-2024` |
| `FT=` | 来源类别 | `FT=CSSCI` |
| `PT=` | 文献类型 | `PT=期刊` |

### 逻辑运算符

| 运算符 | 含义 | 优先级 |
|--------|------|--------|
| `AND` | 并且 | 高 |
| `OR` | 或者 | 中 |
| `AND NOT` | 不含 | 低 |

### 完整示例

```
SU=人工智能 OR SU=大语言模型 OR SU=生成式AI AND NOT SU=综述 AND YE=2020-2024 AND FT=CSSCI
```

**注意**：知网高级检索框中可直接粘贴上述字符串，无需额外调整。

## 2. 万方数据高级检索语法

### 字段标识符

| 标识符 | 含义 | 示例 |
|--------|------|------|
| `主题:` | 主题 | `主题:(人工智能)` |
| `题名:` | 题名 | `题名:(深度学习)` |
| `关键词:` | 关键词 | `关键词:(神经网络)` |
| `摘要:` | 摘要 | `摘要:(自然语言处理)` |
| `作者:` | 作者 | `作者:(张三)` |
| `机构:` | 机构 | `机构:(北京大学)` |
| `年份:` | 年份 | `年份:[2020-2024]` |
| `期刊级别:` | 期刊级别 | `期刊级别:CSSCI` |

### 逻辑运算符

| 运算符 | 含义 |
|--------|------|
| `AND` | 并且 |
| `OR` | 或者 |
| `NOT` | 不含 |

### 完整示例

```
主题:(人工智能 OR 大语言模型 OR 生成式AI) NOT 主题:(综述) AND 年份:[2020-2024] AND 期刊级别:CSSCI
```

## 3. 维普网高级检索语法

### 字段标识符

| 标识符 | 含义 | 示例 |
|--------|------|------|
| `题名或关键词=` | 题名或关键词 | `题名或关键词=人工智能` |
| `题名=` | 题名 | `题名=深度学习` |
| `关键词=` | 关键词 | `关键词=神经网络` |
| `摘要=` | 摘要 | `摘要=自然语言处理` |
| `作者=` | 作者 | `作者=张三` |
| `机构=` | 机构 | `机构=北京大学` |
| `出版年=` | 出版年 | `出版年=2020-2024` |
| `收录类型=` | 收录类型 | `收录类型=CSSCI` |

### 逻辑运算符

| 运算符 | 含义 |
|--------|------|
| `并且` / `*` | 并且 |
| `或者` / `+` | 或者 |
| `不含` / `-` | 不含 |

### 完整示例

```
题名或关键词=人工智能+大语言模型+生成式AI 不含=综述 出版年=2020-2024 收录类型=CSSCI
```

## 4. 同义词扩展规则

### 扩展原则

1. **优先扩展学术界通用术语**：使用学科领域内的标准同义表达。
2. **控制扩展数量**：每组关键词扩展后总数不超过 5 个，避免检索结果过于宽泛。
3. **区分近义词与上下位词**：近义词用 `OR` 连接，上下位词根据用户研究精度决定是否纳入。

### 常用扩展对照表

| 核心词 | 扩展同义词 |
|--------|-----------|
| 人工智能 | 人工智能, 生成式AI, 大语言模型, 机器学习, 深度学习 |
| 高校图书馆 | 高校图书馆, 大学图书馆, 学术图书馆 |
| 智慧服务 | 智慧服务, 智能服务, 精准服务, 个性化服务 |
| 数字人文 | 数字人文, 人文计算, 数字学术 |
| 知识图谱 | 知识图谱, 知识网络, 语义网络 |
| 用户画像 | 用户画像, 读者画像, 用户建模 |
| 信息素养 | 信息素养, 信息能力, 媒介素养 |
| 数据治理 | 数据治理, 数据管理, 数据质量 |

### 扩展后检索式示例

**知网**：
```
SU=高校图书馆 OR SU=大学图书馆 OR SU=学术图书馆 AND SU=智慧服务 OR SU=智能服务 OR SU=精准服务 AND NOT SU=公共图书馆 AND YE=2020-2024
```

## 5. 参考文献输入 JSON Schema

### 基本字段定义

```json
[
  {
    "title": "论文标题（必填）",
    "authors": ["作者1", "作者2"],
    "journal": "期刊名称（必填）",
    "year": 2024,
    "volume": "卷号",
    "issue": "期号",
    "pages": "起始页-结束页",
    "doi": "DOI号（可选）"
  }
]
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `title` | string | 是 | 论文完整标题 |
| `authors` | array[string] | 是 | 作者列表，按原文顺序排列 |
| `journal` | string | 是 | 期刊完整名称 |
| `year` | integer | 是 | 出版年份 |
| `volume` | string | 否 | 卷号 |
| `issue` | string | 否 | 期号 |
| `pages` | string | 否 | 页码范围，如 "45-52" |
| `doi` | string | 否 | DOI标识符 |

### 完整示例

```json
[
  {
    "title": "人工智能在图书馆服务中的应用研究",
    "authors": ["张三", "李四", "王五"],
    "journal": "中国图书馆学报",
    "year": 2023,
    "volume": "49",
    "issue": "3",
    "pages": "45-52",
    "doi": "10.13530/j.cnki.jlis.202303004"
  },
  {
    "title": "Large language models for information retrieval",
    "authors": ["Smith John", "Wang Li"],
    "journal": "Journal of the Association for Information Science and Technology",
    "year": 2024,
    "volume": "75",
    "issue": "2",
    "pages": "112-128",
    "doi": "10.1002/asi.24801"
  }
]
```

## 6. 常见错误与注意事项

### 检索式生成常见错误

1. **字段标识符拼写错误**：知网的 `SU=` 不能写成 `Su=` 或 `su=`，区分大小写。
2. **逻辑运算符混用**：不同平台运算符不同，不能将一个平台的检索式直接用于另一个平台。
3. **年份格式不一致**：知网用 `YE=2020-2024`，万方用 `年份:[2020-2024]`，维普用 `出版年=2020-2024`。
4. **同义词扩展过度**：扩展过多会导致检索结果噪音过大，建议每组核心词扩展不超过 5 个。

### 期刊判断注意事项

1. **目录时效性**：北大核心每 3 年更新一次，CSSCI 每 2 年更新一次，脚本内置数据可能滞后，重要投稿前请核实最新目录。
2. **期刊更名**：部分期刊更名后 ISSN 不变但名称变化，判断时建议同时提供 ISSN。
3. **增刊与正刊**：增刊（Supplement）通常不被认定为核心期刊，需特别说明。

### 参考文献格式注意事项

1. **作者姓名顺序**：GB/T 7714 要求中文文献作者按原文顺序，英文文献姓在前名缩写。
2. **期刊名称缩写**：APA 和 MLA 格式中，英文期刊名需使用标准缩写（脚本目前输出全名，如需标准缩写需额外处理）。
3. **DOI 格式**：GB/T 7714 要求 `DOI:10.xxxx`，APA 要求 `https://doi.org/10.xxxx`，MLA 要求 `doi:10.xxxx`。
